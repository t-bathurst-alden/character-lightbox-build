# Character Lightbox Build
My CSS styling assignment for Authoring class, which is focused around the villains of Jujutsu Kaisen. This is to explore CSS capabilities with characters I greatly enjoy.
## Installation
No download required, just download the ZIP file and extract.
## Usage
Open index.html in whatever browser you wish.
